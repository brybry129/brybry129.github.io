Automated Cookie Clicker

Made with Python using Selenium Web Driver

About the Project

Using the selenium web driver module this project takes the addictive idle game Cookie Clicker to a whole new level of idle. Using the experimental cookie clicker website, https://orteil.dashnet.org/experiments/cookie/, I automated the process to click the cookies and purchase upgrades.

The program runs for 5 minutes and can be modified to run as long as wanted. The upgrades are set up to be purchased every 5 seconds for the first 3 minutes. It will purchase the most expensive upgrade available first and then any others available afterwards. After 3 minutes the time in between upgrade purchases is increased slightly to allow the higher value upgrades to be purchased. These times can also be modified to purchase at any time wanted.
