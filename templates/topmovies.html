List of Top Movies

Made with Python, Flask, FlaskForm and SQL

About the Project

This project is a website built with Flask and SQL to keep a list of my top favorite movies and my review/rating.

Add new movies by searching for the name and selecting from a list. The list of potential movies is retrieved using The Movie Database (TMDB) API. The description and image for each movie entry is also retrieved using the API.

Update a movie's rating and review.

Delete a movie from the list
